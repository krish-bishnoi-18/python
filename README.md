# My Python & DSA Journey 

Welcome to my personal repository! This is a dedicated space where I document my daily coding practice, starting with Python fundamentals and moving into Data Structures and Algorithms (DSA).

##  Topics Covered

###  Phase 1: Python Fundamentals
* **Variables & Data Types**: Strings, integers, floats, booleans.
* **Control Flow**: If-else conditions, match-case.
* **Loops**: For loops, while loops, nested loops.
* **Data Structures**: Lists, tuples, dictionaries, sets.
* **Functions**: Defining functions, parameters, return values, lambda functions.
* **File Handling**: Reading and writing text/CSV files.
* **OOP**: Classes, objects, inheritance, polymorphism.
